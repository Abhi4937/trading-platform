Helper Functions — option_chain_service.py

_years_to_expiry(expiry_dt)

File: option_chain_service.py

Called by: get_option_chain()

Converts expiry datetime to years remaining. Attaches UTC timezone if naive. Clamps to 0.0 for expired options.

Black-Scholes requires T in years to match annualised sigma and r.

expiry_dt: datetime → float (years)

_parse_settlement(product)

File: option_chain_service.py

Called by: get_expiries(), get_option_chain()

Extracts settlement_time from a product dict. Handles both Unix timestamp (int/float) and ISO string formats. Returns None if missing or unparseable.

product: dict → datetime | None

_safe_float(val, default)

File: option_chain_service.py

Called by: _extract_live_greeks(), get_option_chain()

Safely converts any value (string, int, None) to float. Delta API returns numbers as strings. Returns default (0.0) on failure.

val: Any, default: float → float

_extract_live_greeks(ticker)

File: option_chain_service.py

Called by: get_option_chain() — once per strike

Extracts Greeks and IV from raw Delta ticker dict. Reads ticker["greeks"] for delta/gamma/theta/vega/rho. Reads ticker["quotes"] for bid_iv/ask_iv/best_bid/best_ask. Auto-detects IV format (percent ≥2.0 vs decimal <2.0). Returns has_greeks=True if greeks object has delta.

ticker: dict → dict(iv, iv_pct, delta, gamma, theta, vega, rho, bid, ask, oi, has_greeks)

_to_iv_decimal(raw) [inner]

File: option_chain_service.py — inside _extract_live_greeks

Normalises IV to decimal regardless of format. v≥2.0 = percent format → divide by 100. v<2.0 = already decimal. e.g. "49.9"→0.499 "0.499"→0.499 "118.5"→1.185

raw: str|float → float (decimal)

_build_demo_chain(expiry, spot)

File: option_chain_service.py

Called by: get_option_chain() when no products found for expiry

Generates synthetic option chain using Black-Scholes. Creates 21 strikes (ATM ± 10 × $1000). IV smile modelled as quadratic in log-moneyness. Used when no API key or Delta returns nothing.

expiry: date, spot: float → OptionChainResponse

compute_greeks(S, K, T, r, sigma, option_type)

File: core/greeks.py

Called by: get_option_chain(), _build_demo_chain(), plot_data.py

Full Black-Scholes Greeks. Uses Abramowitz & Stegun CDF approximation. theta = per calendar day. vega = per 1% IV move. r=0.0 for crypto convention.

S,K,T,r,sigma,type → Greeks(iv, delta, gamma, theta, vega, rho, price_bs)

implied_vol(market_price, S, K, T, r, option_type, sigma_seed)

File: core/greeks.py

Called by: get_option_chain() — Priority 3 fallback only

Newton-Raphson IV solver. Initial guess = Brenner-Subrahmanyam ATM approx or sigma_seed. Converges in 2-3 iterations with good seed, up to 200 without. Uses mark_price not stale close.

price,S,K,T,r,type,seed → float (IV decimal)

_auth_headers(method, path)

File: delta_client.py — DeltaClient method

Called by: DeltaClient.get() — every single API call

Builds HMAC-SHA256 auth headers. msg = method + timestamp + path. Returns {} if no API key (public endpoints still work).

method:str, path:str → dict(api-key, timestamp, signature)

get_delta_client()

File: delta_client.py — module-level singleton

Called by: get_spot(), get_expiries(), get_option_chain(), plot_data.py

Returns the singleton DeltaClient instance. Creates it on first call. Reuses httpx.AsyncClient with connection pooling (max 20 connections).

() → DeltaClient (singleton)
