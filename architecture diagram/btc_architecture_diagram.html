Core business logic. get_option_chain(): Redis check → parallel Delta fetch → batch ticker requests (20 at a time) → compute Greeks → cache result. Also get_expiries(), get_spot(), _build_demo_chain().

HTTP client for Delta Exchange. HMAC-SHA256 auth. Methods: get_spot_price(), get_btc_option_products(), get_ticker(sym), get_candles(). Singleton via get_delta_client().
app/core/greeks.py
pure math — no external calls
Black-Scholes from scratch. compute_greeks(S,K,T,r,σ) → Greeks dataclass. implied_vol() → Newton-Raphson 200 iterations. _norm_cdf(), _norm_pdf() implemented without scipy.
app/cache/redis_cache.py
calls: redis.asyncio
Async Redis wrapper. init_cache() / close_cache() for lifespan. get(key) → JSON or None. set(key, value, ttl). Silent fallback — all errors caught, returns None.
app/core/config.py

Pydantic BaseSettings. All config: DELTA keys, REDIS_URL, TTLs, CORS_ORIGINS, RISK_FREE_RATE. lru_cache singleton. Extra fields ignored so .env can have comments.

All Pydantic response models. OptionLeg (all Greeks + prices), ChainRow (call+put pair), OptionChainResponse, ExpiryListResponse, SpotResponse, CandleBar, IVSmileResponse, IVRVResponse.

Redis Cache Strategy
1
Request arrives at FastAPI endpoint
2
Check Redis with key like chain:BTC:2025-03-28
cached = await redis_cache.get(cache_key)
✓
Cache HIT → return immediately, Delta API not called, <1ms response
✗
Cache MISS → fetch from Delta Exchange, compute Greeks, store with TTL
spot:BTC → TTL 5s (changes every second)
chain:BTC:{date} → TTL 15s (balance freshness/rate limits)
expiries:BTC → TTL 300s (rarely changes)
candles:{...} → TTL 60s (historical OHLCV)
Black-Scholes Greeks Computation
1
Get market price and spot from Delta ticker
last_price, bid, ask, mark_iv = ticker data
2
Get IV: use exchange mark_iv if available, else Newton-Raphson
iv = float(mark_iv)/100
OR implied_vol(price, S, K, T, r, type)
3
Compute d1, d2 using Black-Scholes formula
d1 = (log(S/K) + (r + σ²/2)T) / (σ√T)
d2 = d1 - σ√T
4
Return Greeks dataclass
delta=N(d1) gamma=n(d1)/(S·σ·√T)
theta=(per day) vega=(per 1% IV)
Delta Exchange Auth — HMAC-SHA256
1
Build message string from method + timestamp + path
msg = "GET" + "1705000000" + "/v2/tickers/BTCUSDT"
2
Sign with API secret using HMAC-SHA256
sig = hmac.new(API_SECRET, msg, sha256).hexdigest()
3
Attach to request headers
headers = {"api-key": KEY, "timestamp": ts, "signature": sig}
4
No API key → empty headers → public endpoints work in demo mode
